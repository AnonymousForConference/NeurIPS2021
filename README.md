# NeurIPS2021
#### Reproducing experimental results of Paper Id 3052

## Requirements
#### python >= 3.7.3
#### torch >= 1.1.0
#### torchvision >= 0.2.2

## To Run the Codes
#### python main.py --data Cifar10 --method LADA
- Data will be downloaded to folder 'data'.
- Result will be recorded to foler 'Results'.
